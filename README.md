Hi, I'm Farel! A dedicated Full-Stack Developer and student at SMK Taruna Bhakti. My journey in software development is driven by a passion for creating efficient, scalable, and user-centric digital solutions. I thrive on bridging the gap between complex backend logic and seamless frontend experiences, consistently exploring new methodologies to write cleaner, more performant code.
